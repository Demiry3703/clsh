## ⚙️ How clsh Works

clsh uses the following technologies:

- **SSH Protocol:** For secure remote connection to terminals
- **WebSocket:** To provide a live terminal experience in your browser or app
- **tmux Integration:** Allows multiple terminal sessions in one window
- **Node-pty:** Provides full support for terminal commands on the backend
- **Typescript:** Ensures stable and secure code in the app
- **PWA Features:** Make the interface installable on phones and tablets

## 🔎 Troubleshooting Tips

- If you cannot connect, check if your PC is online and accessible.

- Make sure the IP address or host name you enter is correct.

- Verify your SSH username and password.

- Ensure your Windows firewall allows clsh to communicate.

- Use a modern, up-to-date browser for mobile connections.

- If clsh crashes or does not start, try reinstalling after closing other running programs.

- Restart your PC if connections seem unstable.
